# 第08章：卫星相对运动计算

本章学习如何使用 Orekit 计算两颗卫星的相对运动（Hill/LVLH 坐标系）。

## 对应代码文件

| 文件 | 说明 |
|------|------|
| `08_relative_motion.py` | Hill 坐标系变换、相对状态计算、多周期相对运动轨迹可视化 |

## 8.1 Hill 坐标系（LVLH）定义

Hill 坐标系是以 Chief 卫星为原点的局部坐标系：

| 轴 | 方向 | 说明 |
|----|------|------|
| **x** | 径向 | 从地心指向卫星 |
| **y** | 横向（沿迹向） | 沿轨道运动方向 |
| **z** | 法向 | 垂直于轨道面 |

## 8.2 计算流程

```
1. 获取 Chief 和 Deputy 在惯性系中的位置/速度
           ↓
2. 计算惯性系差值：Δr = r_deputy - r_chief, Δv = v_deputy - v_chief
           ↓
3. 使用 313 欧拉角旋转矩阵变换到 Hill 坐标系
           ↓
4. 减去牵连速度 [-n*y, n*x, 0]
           ↓
5. 得到 Hill 相对状态 [x, y, z, vx, vy, vz]
```

## 8.3 关键公式

### 旋转矩阵（313 欧拉角）

```python
A = Au @ Ai @ AOMG

# Au: 升交点角距旋转 (绕 z 轴)
Au = [[cos(u), sin(u), 0],
      [-sin(u), cos(u), 0],
      [0, 0, 1]]

# Ai: 轨道倾角旋转 (绕 x 轴)
Ai = [[1, 0, 0],
      [0, cos(i), sin(i)],
      [0, -sin(i), cos(i)]]

# AOMG: 升交点赤经旋转 (绕 z 轴)
AOMG = [[cos(Omega), sin(Omega), 0],
        [-sin(Omega), cos(Omega), 0],
        [0, 0, 1]]

# u = omega + f (升交点角距)
```

### 牵连速度

```python
# Hill 坐标系中的牵连速度（旋转效应）
transport = [-n * y, n * x, 0]

# 其中 n = sqrt(mu / a^3) 是轨道角速度
```

## 8.5 Clohessy-Wiltshire 方程

对于圆轨道，相对运动满足 CW 方程：

```
x(t) = (4-3cos(nt))*x0 + (sin(nt)/n)*vx0 + (2/n)*(1-cos(nt))*vy0
y(t) = 6*(sin(nt)-nt)*x0 + (2/n)*(cos(nt)-1)*vx0 + (4*sin(nt)-3*nt)*y0 + (vy0/n)*(1-cos(nt))
z(t) = z0*cos(nt) + (vz0/n)*sin(nt)
```
