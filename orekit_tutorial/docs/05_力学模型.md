## 5.1 力学模型概览

轨道传播时，除了中心引力外，还可以加入各种摄动力：

| 力学模型 | 类 | 说明 | 适用场景 |
|---------|-----|------|---------|
| 中心引力 | `NewtonianAttraction` | 两体问题基本力 | 所有轨道 |
| 地球重力场 | `HolmesFeatherstoneAttractionModel` | 高阶球谐展开 | 精确轨道预报 |
| 第三体引力 | `ThirdBodyAttraction` | 日月等天体引力 | 高轨/长期预报 |
| 大气阻力 | `DragForce` | 低轨卫星主要耗散力 | LEO 轨道衰减 |
| 太阳光压 | `SolarRadiationPressure` | 高轨卫星主要摄动力 | GEO/MEO 轨道 |

## 5.2 中心引力

最基本的力学模型，即两体问题：

```python
from org.orekit.forces.gravity import NewtonianAttraction

mu = Constants.EGM96_EARTH_MU
propagator.addForceModel(NewtonianAttraction(mu))
```

## 5.3 地球重力场（Holmes-Featherstone）

使用球谐展开描述地球非球形引力：

```python
from org.orekit.forces.gravity import HolmesFeatherstoneAttractionModel
from org.orekit.forces.gravity.potential import GravityFieldFactory

# 加载重力场（阶数越高越精确，计算量越大）
provider = GravityFieldFactory.getNormalizedProvider(6, 6)  # 6x6 阶

# 创建力学模型
itrf = FramesFactory.getITRF(IERSConventions.IERS_2010, True)
hf_model = HolmesFeatherstoneAttractionModel(itrf, provider)
propagator.addForceModel(hf_model)
```

常用重力场模型：

| 模型 | 阶数 | 适用场景 |
|------|------|---------|
| 2x2 | 最快 | 粗略估计 |
| 6x6 | 中等 | 一般任务 |
| 20x20 | 较慢 | 精确轨道确定 |
| 70x70 | 很慢 | 高精度科学任务 |

## 5.4 第三体引力

太阳、月球等天体的引力摄动：

```python
from org.orekit.forces.gravity import ThirdBodyAttraction
from org.orekit.bodies import CelestialBodyFactory

# 添加太阳引力
sun = CelestialBodyFactory.getSun()
propagator.addForceModel(ThirdBodyAttraction(sun))

# 添加月球引力
moon = CelestialBodyFactory.getMoon()
propagator.addForceModel(ThirdBodyAttraction(moon))
```

## 5.5 大气阻力

低地球轨道（LEO）的主要耗散力：

```python
from org.orekit.forces.drag import DragForce, IsotropicDrag
from org.orekit.models.earth.atmosphere import HarrisPriester
from org.orekit.bodies import OneAxisEllipsoid

# 创建大气模型
earth = OneAxisEllipsoid(
    Constants.WGS84_EARTH_EQUATORIAL_RADIUS,
    Constants.WGS84_EARTH_FLATTENING, itrf
)
sun = CelestialBodyFactory.getSun()
atmosphere = HarrisPriester(sun, earth)

# 创建航天器形状（球形）
sc = IsotropicDrag(0.01, 2.2)  # 截面积 0.01 m², Cd = 2.2

# 创建阻力模型
drag_force = DragForce(atmosphere, sc)
propagator.addForceModel(drag_force)
```

## 5.6 太阳光压

高轨道卫星的主要摄动力：

```python
from org.orekit.forces.radiation import SolarRadiationPressure, IsotropicRadiationSingleCoefficient

# 创建航天器光压模型
# 参数：有效截面积 (m^2), 光压系数 Cr（典型值 1.0-2.0）
sc = IsotropicRadiationSingleCoefficient(0.01, 1.5)

# 创建光压模型
# 注意：构造函数签名是 (sun, equatorialRadius, spacecraft)
# 第二个参数是地球赤道半径（遮挡天体的半径），不是天体对象
sun = CelestialBodyFactory.getSun()
earth_radius = Constants.WGS84_EARTH_EQUATORIAL_RADIUS
srp = SolarRadiationPressure(sun, earth_radius, sc)
propagator.addForceModel(srp)
```

## 5.7 组合力学模型

可以同时添加多种力学模型：

```python
from org.orekit.forces.radiation import SolarRadiationPressure, IsotropicRadiationSingleCoefficient

# 中心引力 + 重力场 + 日月引力 + 太阳光压
propagator.addForceModel(NewtonianAttraction(mu))
propagator.addForceModel(HolmesFeatherstoneAttractionModel(itrf, provider))
propagator.addForceModel(ThirdBodyAttraction(sun))
propagator.addForceModel(ThirdBodyAttraction(moon))

# 太阳光压（注意构造函数参数）
sc_srp = IsotropicRadiationSingleCoefficient(0.01, 1.5)
earth_radius = Constants.WGS84_EARTH_EQUATORIAL_RADIUS
srp = SolarRadiationPressure(sun, earth_radius, sc_srp)
propagator.addForceModel(srp)
```

## 5.8 各力学模型的影响

| 力学模型 | 主要影响的轨道参数 |
|---------|-------------------|
| J2 重力场 | 升交点进动、近地点进动 |
| 大气阻力 | 半长轴衰减、离心率变化 |
| 日月引力 | 轨道面长期变化 |
| 太阳光压 | 离心率长期变化 |
