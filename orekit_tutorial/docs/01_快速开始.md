## 1.1 Orekit 是什么

Orekit 是一个开源的航天动力学 Java 库，由法国航天局 (CNES) 开发维护。它提供了：

- 轨道创建与表示（开普勒要素、笛卡尔坐标等）
- 轨道传播（解析法、数值法）
- 力学模型（地球引力、大气阻力、太阳光压等）
- 坐标系转换
- TLE（两行根数）解析与传播

我们通过 Python 包装器（JPype）在 Python 中调用 Orekit。

## 1.2 初始化 Orekit

每个 Orekit 程序的第一步都是初始化：

```python
import orekit
orekit.initVM()
```

这行代码启动 Java 虚拟机 (JVM)，之后就可以使用 Orekit 的所有功能。

## 1.3 加载数据

Orekit 依赖外部数据文件（地球定向参数、行星历表等）。在使用任何 Orekit 功能之前，必须先加载数据：

```python
from java.io import File
from org.orekit.data import DataContext, DirectoryCrawler

dm = DataContext.getDefault().getDataProvidersManager()
data_dir = File(r"D:\orekit-data-main")  # 改为你的数据路径
dm.clearProviders()
dm.clearLoadedDataNames()
dm.resetFiltersToDefault()
dm.addProvider(DirectoryCrawler(data_dir))
```

> **注意**：本教程使用公共初始化模块 `orekit_init.py`，已封装了上述步骤，后续示例只需 `from orekit_init import *` 即可。

## 1.4 创建你的第一个轨道

下面我们创建一个国际空间站 (ISS) 的近似轨道：

```python
import math
from orekit_init import *

# 定义时间
date = create_date(2024, 1, 1, 12, 0, 0)

# 创建开普勒轨道
# ISS 参数：高度约400km，倾角51.6度
mu = Constants.EGM96_EARTH_MU
orbit = KeplerianOrbit(
    6_778_000.0,            # 半长轴 a (m) = 地球半径6378km + 高度400km
    0.001,                  # 离心率 e（近圆轨道）
    math.radians(51.6),     # 倾角 i (rad)
    0.0,                    # 近地点幅角 ω (rad)
    0.0,                    # 升交点赤经 Ω (rad)
    0.0,                    # 真近点角 ν (rad)
    PositionAngle.TRUE,     # 角度类型：真近点角
    FramesFactory.getGCRF(),# 参考坐标系：地心惯性系
    date,                   # 历元时间
    mu                      # 地球引力常数
)

# 打印轨道参数
print_orbit(orbit, "国际空间站（近似）轨道")
```

## 1.5 关键概念速览

| 概念 | 说明 |
|------|------|
| `AbsoluteDate` | Orekit 中的时间表示 |
| `KeplerianOrbit` | 用六个开普勒要素描述的轨道 |
| `PositionAngle.TRUE` | 真近点角（卫星在轨道上的位置） |
| `FramesFactory.getGCRF()` | 地心惯性参考系 (GCRF) |
| `Constants.EGM96_EARTH_MU` | 地球引力常数 μ = 398600.4415 km³/s² |
