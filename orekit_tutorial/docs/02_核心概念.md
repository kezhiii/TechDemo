## 2.1 时间系统

Orekit 使用 `AbsoluteDate` 表示时间点。创建时间需要指定**时间尺度**：

| 时间尺度 | 说明 | 用途 |
|---------|------|------|
| **UTC** | 协调世界时 | 日常使用，含闰秒 |
| **TAI** | 国际原子时 | 内部计算，无闰秒 |
| **TT** | 地球时 | 行星运动计算 |

```python
from org.orekit.time import AbsoluteDate, TimeScalesFactory, DateComponents, TimeComponents

utc = TimeScalesFactory.getUTC()
date = AbsoluteDate(
    DateComponents(2024, 1, 1),     # 年月日
    TimeComponents(12.0 * 3600.0),   # 一天中的秒数（12小时 = 43200秒）
    utc
)

print(date.toString(utc))  # 输出: 2024-01-01T12:00:00.000Z
```

### 便捷函数

教程提供了 `create_date()` 函数简化时间创建：

```python
date = create_date(2024, 1, 1, 12, 0, 0)          # 默认 UTC
date = create_date(2024, 1, 1, 12, 0, 0, "TAI")   # 指定 TAI
```

### 时间偏移

```python
# 向前偏移 1 小时
date2 = date.shiftedBy(3600.0)  # 单位：秒

# 向前偏移 1 天
date3 = date.shiftedBy(86400.0)
```

## 2.2 坐标系

Orekit 中的坐标系形成一棵**树**，根节点是 GCRF（地心惯性系）。

### 惯性系（不随地球自转）

| 坐标系 | 获取方式 | 说明 |
|--------|---------|------|
| **GCRF** | `FramesFactory.getGCRF()` | 地心惯性参考系，最常用 |
| **EME2000** | `FramesFactory.getEME2000()` | J2000 平赤道面 |
| **ICRF** | `FramesFactory.getICRF()` | 国际天球参考系 |

### 地固系（随地球自转）

| 坐标系 | 获取方式 | 说明 |
|--------|---------|------|
| **ITRF** | `FramesFactory.getITRF(conventions, eop)` | 国际地球参考系 |
| **TEME** | `FramesFactory.getTEME()` | TLE 使用的坐标系 |

### ITRF 创建（需要参数）

```python
from org.orekit.utils import IERSConventions

itrf = FramesFactory.getITRF(IERSConventions.IERS_2010, True)
# 参数：IERS 约定版本, 是否考虑季节变化
```

### 坐标系转换

```python
from org.hipparchus.geometry.euclidean.threed import Vector3D

# 位置向量
pos_gcrf = Vector3D(6771000.0, 0.0, 0.0)

# GCRF -> ITRF
gcrf = FramesFactory.getGCRF()
itrf = FramesFactory.getITRF(IERSConventions.IERS_2010, True)

# 获取变换
transform = gcrf.getTransformTo(itrf, date)
pos_itrf = transform.transformPosition(pos_gcrf)
```

## 2.3 轨道六要素

描述一个椭圆轨道需要 6 个参数：

```
        升交点赤经 Ω (RAAN)
              |
              |    近地点幅角 ω
              |   /
              |  /
              | / 真近点角 ν
              |/
    ─────────●───────── 赤道面
             /|
            / |
           /  |
          /   |
         /    |
        /     |
```

| 要素 | 符号 | 单位 | 含义 |
|------|------|------|------|
| 半长轴 | a | km | 轨道大小 |
| 离心率 | e | 无量纲 | 轨道形状（0=圆，0~1=椭圆） |
| 倾角 | i | deg | 轨道面与赤道面的夹角 |
| 近地点幅角 | ω | deg | 近地点在轨道面内的位置 |
| 升交点赤经 | Ω | deg | 轨道面在空间中的朝向 |
| 真近点角 | ν | deg | 卫星在轨道上的位置 |

### 轨道类型与周期

$$T = 2\pi \sqrt{\frac{a^3}{\mu}}$$

- **LEO**（低地球轨道）：a ≈ 6700-7000 km，T ≈ 90-100 分钟
- **MEO**（中地球轨道）：a ≈ 26000 km，T ≈ 12 小时
- **GEO**（地球同步轨道）：a ≈ 42164 km，T = 24 小时
