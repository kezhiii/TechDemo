# 第03章：轨道创建与表示

本章学习如何创建不同类型的轨道，以及它们之间的相互转换。

## 对应代码文件

| 文件 | 说明 |
|------|------|
| `03_orbit_creation.py` | KeplerianOrbit/CircularOrbit/CartesianOrbit 创建与互转、轨道根数↔惯性系坐标转换 |

## 3.1 轨道类型概览

Orekit 提供三种主要的轨道表示：

| 类型 | 参数 | 适用场景 |
|------|------|---------|
| `KeplerianOrbit` | a, e, i, ω, Ω, ν | 经典六要素，直观易理解 |
| `CircularOrbit` | a, ex, ey, i, Ω, (ω+ν) | 近圆轨道，避免 ω 退化 |
| `CartesianOrbit` | X, Y, Z, Vx, Vy, Vz | 笛卡尔坐标，数值计算方便 |

## 3.2 创建开普勒轨道

```python
import math
from orekit_init import *

date = create_date(2024, 1, 1, 12, 0, 0)
mu = Constants.EGM96_EARTH_MU

# 创建一个椭圆轨道
orbit = KeplerianOrbit(
    7_000_000.0,            # 半长轴 a (m)
    0.01,                   # 离心率 e
    math.radians(98.0),     # 倾角 i (rad) - 太阳同步轨道
    math.radians(90.0),     # 近地点幅角 ω (rad)
    math.radians(0.0),      # 升交点赤经 Ω (rad)
    math.radians(0.0),      # 真近点角 ν (rad)
    PositionAngle.TRUE,
    FramesFactory.getGCRF(),
    date,
    mu
)

print_orbit(orbit, "太阳同步轨道")
```

## 3.3 创建圆轨道

对于近圆轨道（e ≈ 0），使用 `CircularOrbit` 更稳定：

```python
# CircularOrbit 使用偏心率分量代替 e 和 ω
# ex = e * cos(ω + Ω)
# ey = e * sin(ω + Ω)

orbit_circ = CircularOrbit(
    42_164_000.0,           # 半长轴 a (m) - GEO 轨道
    0.0,                    # ex 分量
    0.0,                    # ey 分量
    math.radians(0.01),     # 倾角 i (rad)
    math.radians(0.0),      # 升交点赤经 Ω (rad)
    0.0,                    # 纬度幅角 (ω + ν)
    PositionAngle.TRUE,
    FramesFactory.getGCRF(),
    date,
    mu
)

print_orbit(orbit_circ, "地球同步轨道")
```

## 3.4 创建笛卡尔轨道

直接使用位置和速度向量：

```python
from org.hipparchus.geometry.euclidean.threed import Vector3D
from org.orekit.utils import PVCoordinates

# 位置 (m)
pos = Vector3D(42_164_000.0, 0.0, 0.0)

# 速度 (m/s) - GEO 轨道速度约 3.07 km/s
vel = Vector3D(0.0, 3_070.0, 0.0)

pv = PVCoordinates(pos, vel)

orbit_cart = CartesianOrbit(
    pv,
    FramesFactory.getGCRF(),
    date,
    mu
)

print_orbit(orbit_cart, "笛卡尔轨道表示")
```

## 3.5 轨道类型转换

不同表示之间可以自由转换：

```python
# Keplerian -> Cartesian
orbit_kep = KeplerianOrbit(a, e, i, omega, raan, anomaly,
                           PositionAngle.TRUE, gcrf, date, mu)

# 转换为笛卡尔坐标
pv = orbit_kep.getPVCoordinates()
pos = pv.getPosition()
vel = pv.getVelocity()
print(f"位置: ({pos.getX()/1000:.2f}, {pos.getY()/1000:.2f}, {pos.getZ()/1000:.2f}) km")
print(f"速度: ({vel.getX()/1000:.3f}, {vel.getY()/1000:.3f}, {vel.getZ()/1000:.3f}) km/s")

# Cartesian -> Keplerian
orbit_cart = CartesianOrbit(pv, gcrf, date, mu)
print(f"半长轴: {orbit_cart.getA()/1000:.3f} km")
print(f"离心率: {orbit_cart.getE():.6f}")
```

## 3.6 轨道参数的物理意义

### 半长轴 a

- 决定轨道的**大小**和**周期**
- 地球半径 Re ≈ 6378 km
- LEO 高度 400km → a ≈ 6778 km，周期 ≈ 92 分钟
- GEO 高度 35786km → a ≈ 42164 km，周期 = 24 小时

### 离心率 e

- 决定轨道的**形状**
- e = 0：圆轨道
- 0 < e < 1：椭圆轨道
- e = 1：抛物线轨道
- e > 1：双曲线轨道

### 倾角 i

- 轨道面与赤道面的夹角
- i = 0°：赤道轨道
- i = 90°：极地轨道
- i ≈ 97.4°：太阳同步轨道（LEO）

### 近地点幅角 ω

- 近地点相对于升交点的角度
- ω = 0°：近地点在升交点处
- ω = 90°：近地点在北极方向

### 升交点赤经 Ω

- 升交点在赤道面上的位置
- 决定轨道面的**空间朝向**

### 真近点角 ν

- 卫星当前位置相对于近地点的角度
- ν = 0°：近地点
- ν = 180°：远地点
