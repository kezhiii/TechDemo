## 6.1 坐标系树结构

Orekit 中的坐标系形成一棵树：

```
GCRF (地心惯性系)
├── EME2000
├── ICRF
└── ITRF (地固系)
    ├── TEME
    └── 局部坐标系
```

根节点是 GCRF（地心惯性参考系），其他坐标系都是它的子节点。

## 6.2 惯性系之间的转换

```python
from orekit_init import *

date = create_date(2024, 1, 1, 12, 0, 0)

# 获取坐标系
gcrf = FramesFactory.getGCRF()
eme2000 = FramesFactory.getEME2000()

# 位置向量
pos_gcrf = Vector3D(6_771_000.0, 0.0, 0.0)

# GCRF -> EME2000
transform = gcrf.getTransformTo(eme2000, date)
pos_eme2000 = transform.transformPosition(pos_gcrf)

print(f"GCRF: ({pos_gcrf.getX()/1000:.3f}, {pos_gcrf.getY()/1000:.3f}, {pos_gcrf.getZ()/1000:.3f}) km")
print(f"EME2000: ({pos_eme2000.getX()/1000:.3f}, {pos_eme2000.getY()/1000:.3f}, {pos_eme2000.getZ()/1000:.3f}) km")
```

## 6.3 惯性系与地固系转换

GCRF（惯性系）与 ITRF（地固系）之间的转换需要考虑地球自转：

```python
from org.orekit.utils import IERSConventions

# 创建 ITRF（需要 IERS 约定版本）
itrf = FramesFactory.getITRF(IERSConventions.IERS_2010, True)

# GCRF -> ITRF
transform = gcrf.getTransformTo(itrf, date)
pos_itrf = transform.transformPosition(pos_gcrf)
vel_itrf = transform.transformVelocity(pos_gcrf, vel_gcrf)

print(f"ITRF: ({pos_itrf.getX()/1000:.3f}, {pos_itrf.getY()/1000:.3f}, {pos_itrf.getZ()/1000:.3f}) km")
```

## 6.4 站心坐标系 (ENU)

以地面站为中心的局部坐标系：

- **E** (East): 东向
- **N** (North): 北向
- **U** (Up): 天顶方向

```python
from org.orekit.bodies import OneAxisEllipsoid

# 创建地球椭球体
earth = OneAxisEllipsoid(
    Constants.WGS84_EARTH_EQUATORIAL_RADIUS,
    Constants.WGS84_EARTH_FLATTENING,
    FramesFactory.getITRF(IERSConventions.IERS_2010, True)
)

# 定义地面站（北京，经度 116.4°E，纬度 39.9°N）
from org.hipparchus.geometry.euclidean.threed import SphericalCoordinates
import math

lat = math.radians(39.9)   # 纬度
lon = math.radians(116.4)  # 经度

# 创建站心坐标系
station_frame = TopocentricFrame(earth, lat, lon, "Beijing")

# 转换到站心坐标系
transform = itrf.getTransformTo(station_frame, date)
pos_topo = transform.transformPosition(pos_itrf)

# ENU 坐标
e = pos_topo.getX()  # 东向
n = pos_topo.getY()  # 北向
u = pos_topo.getZ()  # 天顶

print(f"东向: {e/1000:.3f} km")
print(f"北向: {n/1000:.3f} km")
print(f"天顶: {u/1000:.3f} km")
```

## 6.5 从轨道获取位置并转换

```python
# 创建轨道
orbit = KeplerianOrbit(
    6_778_000.0, 0.001,
    math.radians(51.6), 0.0, 0.0, 0.0,
    PositionAngle.TRUE, gcrf, date, mu
)

# 获取 GCRF 位置
pv = orbit.getPVCoordinates()
pos_gcrf = pv.getPosition()

# 转换到 ITRF
transform = gcrf.getTransformTo(itrf, date)
pos_itrf = transform.transformPosition(pos_gcrf)

# 转换到经纬度
point = earth.transform(pos_itrf, itrf, date)
lat_deg = math.degrees(point.getLatitude())
lon_deg = math.degrees(point.getLongitude())
alt = point.getAltitude()

print(f"纬度: {lat_deg:.4f} deg")
print(f"经度: {lon_deg:.4f} deg")
print(f"高度: {alt/1000:.3f} km")
```
